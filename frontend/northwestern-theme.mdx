The Northwestern Laravel Starter includes built-in support for Northwestern University's brand guidelines, including colors, typography, logos, and layout patterns.

## Brand Configuration

The Northwestern theme is configured in `config/northwestern-theme.php`:

```php
return [
    // Brand lockup image
    'lockup' => env('NU_THEME_LOCKUP', 
        'https://common.northwestern.edu/v8/css/images/northwestern.svg'),

    // Office contact information
    'office' => [
        'name' => env('NU_THEME_OFFICE_NAME', 'Information Technology'),
        'addr' => env('NU_THEME_OFFICE_ADDR', '1800 Sherman Ave'),
        'city' => env('NU_THEME_OFFICE_CITY', 'Evanston, IL 60201'),
        'phone' => env('NU_THEME_OFFICE_PHONE', '847-491-4357 (1-HELP)'),
        'email' => env('NU_THEME_OFFICE_EMAIL', 'consultant@northwestern.edu'),
    ],

    // Global alerts
    'globalAlerts' => [
        App\Domains\Core\GlobalAlerts\UserImpersonated::class,
    ],
];
```

## Environment Configuration

Customize branding for your department by setting environment variables:

```bash
# .env
NU_THEME_LOCKUP=https://your-department.northwestern.edu/logo.svg
NU_THEME_OFFICE_NAME="Kellogg School of Management"
NU_THEME_OFFICE_ADDR="2211 Campus Drive"
NU_THEME_OFFICE_CITY="Evanston, IL 60208"
NU_THEME_OFFICE_PHONE="847-491-3300"
NU_THEME_OFFICE_EMAIL="kellogg-info@northwestern.edu"
```

<Note>
The default lockup uses the Northwestern wordmark. Replace it with your department or school's approved lockup image.
</Note>

## Northwestern Colors

The theme uses Northwestern's official color palette defined in `resources/sass/_variables.scss`:

```scss
// Northwestern Purple
$purple: #4e2a84 !default;          // Primary brand color
$purple-dark: #401f68 !default;     // Darker shade for hover states
$purple-light: #dcd4e6 !default;    // Light shade for backgrounds

// Additional Northwestern colors

$teal: #7fcecd;
```

## Typography

### Northwestern Fonts

The application uses Northwestern's brand typography loaded from `resources/sass/nu_fonts.scss`:

**Akkurat Pro** - Primary typeface
- Akkurat Pro Regular (body text)
- Akkurat Pro Bold (emphasis and links)

**Font Stack**:
```scss
$font-family-sans-serif: "Akkurat Pro Regular", 
    -apple-system, BlinkMacSystemFont, 
    "Segoe UI", Roboto, "Helvetica Neue", Arial, 
    sans-serif;
```

**Alternative Fonts**:
- **Poppins Bold**: Used for special headings
- **Poppins Light**: Used for lighter text

### Font Usage

```html
<p>Body text uses Akkurat Pro Regular</p>
<a href="#">Links use Akkurat Pro Bold</a>
<h1 class="font-poppins-bold">Poppins Bold heading</h1>
<p class="font-poppins-light">Poppins Light text</p>
```

## Northwestern Lockup

### Default Lockup

The default Northwestern wordmark is loaded from:
```
https://common.northwestern.edu/v8/css/images/northwestern.svg
```

### Custom Department Lockup

For department-specific applications, use your approved lockup:

**Option 1: Environment Variable**
```bash
NU_THEME_LOCKUP="https://your-dept.northwestern.edu/lockup.svg"
```

**Option 2: Local Asset**
```bash
# Place lockup in public directory
NU_THEME_LOCKUP="/images/department-lockup.svg"
```

### Lockup Specifications

Follow Northwestern's brand guidelines:
- Use approved lockup images only
- Maintain proper aspect ratio (typically 358px Ã— 42px)
- Ensure adequate clear space around the logo
- Use SVG format for scalability

## Filament Theme Integration

The Filament admin panel uses Northwestern colors:

**Panel Configuration** (`app/Providers/Filament/AdminPanelProvider.php`):
```php
use Filament\Support\Colors\Color;

public function panel(Panel $panel): Panel
{
    return $panel
        ->colors([
            'primary' => Color::hex('#4E2A84'),
        ])
        ->brandLogo(config('northwestern-theme.lockup'))
        ->brandName(config('app.name'));
}
```

**Custom Theme CSS** (`resources/css/filament/administration/theme.css`):
```css
@import '/vendor/filament/filament/resources/css/theme.css';

:root {
    --primary: 78 42 132; /* Northwestern Purple in RGB */
}
```

```scss
.btn-northwestern {

        $border: $purple,
```
