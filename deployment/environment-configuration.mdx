## Overview

This page provides a comprehensive reference for all environment variables used in Northwestern Laravel Starter. All variables are configured in your `.env` file.

## Core Application

### Application Settings

<ParamField path="APP_NAME" type="string" default="Northwestern Laravel Starter">
  Application name displayed throughout the interface
</ParamField>

<ParamField path="APP_ENV" type="string" default="local" required>
  Application environment: `local`, `staging`, `production`, `testing`, or `ci`
</ParamField>

<ParamField path="APP_KEY" type="string" required>
  Encryption key for securing session data and encrypted values. Generate with `php artisan key:generate`
</ParamField>

<ParamField path="APP_DEBUG" type="boolean" default="true">
  Enable debug mode. **Must be `false` in production**
</ParamField>

<ParamField path="APP_URL" type="string" default="http://localhost">
  The URL where your application is accessible
</ParamField>

<ParamField path="PRODUCTION_URL" type="string">
  Canonical production URL used by non-production environments for redirects and links
</ParamField>

### Localization

<ParamField path="APP_LOCALE" type="string" default="en">

<ParamField path="APP_FALLBACK_LOCALE" type="string" default="en">

<ParamField path="APP_FAKER_LOCALE" type="string" default="en_US">

<ParamField path="DEFAULT_USER_TIMEZONE" type="string" default="America/Chicago">

## Object Storage (MinIO/S3)

<ParamField path="AWS_ACCESS_KEY_ID" type="string" required>
  Access key ID for MinIO or AWS S3
</ParamField>

<ParamField path="AWS_SECRET_ACCESS_KEY" type="string" required>
  Secret access key for MinIO or AWS S3
</ParamField>

<ParamField path="AWS_DEFAULT_REGION" type="string" default="us-east-1">
  AWS region
</ParamField>

<ParamField path="AWS_URL" type="string" required>
  Public URL for accessing stored objects
</ParamField>

<ParamField path="AWS_ENDPOINT" type="string" required>
  S3-compatible endpoint URL
</ParamField>

<ParamField path="AWS_USE_PATH_STYLE_ENDPOINT" type="boolean" default="true">
  Use path-style URLs (required for MinIO)
</ParamField>

<ParamField path="AWS_BUCKET" type="string" required>
  Bucket name for file storage
</ParamField>

<ParamField path="AWS_MINIO_CONSOLE" type="string">
  MinIO console URL for local development
</ParamField>

<ParamField path="S3_STREAM_READS" type="boolean" default="false">
  Enable streaming for large file reads to reduce memory usage
</ParamField>

<Warning>
  When enabling `S3_STREAM_READS`, ensure your HTTP client can connect to the endpoint. cURL may have issues with self-signed certificates.
</Warning>

### Environment Lockdown

<ParamField path="ENVIRONMENT_LOCKDOWN_ENABLED" type="boolean">
  Restrict access to users with non-default roles. Defaults to `true` for staging, `false` for production/local/testing/ci
</ParamField>

<Note>
  Environment lockdown helps protect staging and demo environments from unauthorized access while allowing authenticated users in production.
</Note>
